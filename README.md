# IBRD Debt Visualization

Interactive visualizations of World Bank debt trajectories from 1946 to 2025.
[Click here to explore](https://somdeepkundu.github.io/ibrd-debt-visualization/IBRD_VelocityMap_v3_responsive.html)

## Visualizations

- **Interactive Timeline** - Explore debt trajectories across time
- **Time Series Chart** - Track debt changes over the years
- **Slope Interactive** - Compare debt trends between countries
- **Velocity Map** - Visualize rate of change in debt
- **Shifts Analysis** - Analyze 20-year debt shifts

## View the Visualizations

Open any of the HTML files in your browser to explore the interactive visualizations:

- `IBRD_Interactive_Timeline_1946-2025.html`
- `IBRD_TimeSeries_Chart.html`
- `IBRD_Slope_Interactive.html`
- `IBRD_VelocityMap.html` or `IBRD_VelocityMap_v3.html`
- `ibrd_shifts_20_years.html`

## Data Source

World Bank International Bank for Reconstruction and Development (IBRD) data
