🗺️ Velocity Map v3 (Advanced)

Slope-based debt trajectory analysis with interactive segment details

✅ Advanced Slope Analysis: Each segment shows From/To values, change amount (Δ), and slope in Cr/year. Interactive: Hover over any line segment to see detailed breakdown. Click legend to toggle countries.
📊 How to read:
Each colored line represents one country's IBRD debt trajectory. Circular nodes mark data points. Hover over any line segment to see:
• Period start and end years
• From/To debt values in millions (converted to Cr)
• Change (Δ) in that period
• Slope = growth rate per year (Cr/yr) with description (rapidly rising/gently rising/stable/declining)
Click legend items to toggle countries on/off
